Structure in c

Structure is a group of variables of different data types represented by a single name. Lets take an example to understand the need of a structure in C programming.
How to create a structure in C Programming We use struct keyword to create a structure in C. The struct keyword is a short form of structured data type.

struct struct_name {
DataType member1_name;
DataType member2_name;
DataType member3_name;
…
};

* Here struct_name can be anything of your choice. Members data type can be same or different. Once we have declared the structure we can use the struct name as a data type like int, float etc.
* Example of Structure in C.
Input :
#include<stdio.h>
#include<conio.h>
/* Created a structure here. The name of the structure is
* StudentData.
*/
struct StudentData
{
char *stu_name;
int stu_id;
int stu_age;
};
void main()
{
clrscr();
/* student is the variable of structure StudentData*/
struct StudentData student;
/*Assigning the values of each struct member here*/
student.stu_name = "Steve";
student.stu_id = 1234;
student.stu_age = 30;
/* Displaying the values of struct members */
printf("Student Name is: %s", student.stu_name);
printf("\nStudent Id is: %d", student.stu_id);
printf("\nStudent Age is: %d", student.stu_age);
getch();
}
Output :
Student Name is: Steve
Student Id is: 1234
Student Age is: 30

Program to Store Information of Students Using Structure

In this program, a structure, student is created. This structure has three members: name (string), roll (integer) and marks (float). Then, we created a structure array of size 10 to store information of 10 students. Using for loop, the program takes the information of 10 students from the user and displays it on the screen.
Input :
#include<stdio.h>
#include<conio.h>

struct student
{
char name[50];
int roll;
int marks;
};
void main()
{
student s[2];
int i;
clrscr();
printf("\n Enter information of students:");
for(i=0;i<2;i++)
{
s[i].roll=i+1;
printf("\n For roll no %d",s[i].roll);
printf("\n enter name : ");
scanf("%s",&s[i].name);
printf("\n enter marks : ");
scanf("%d",&s[i].marks);
}
for(i=0;i<2;i++)
{
printf("\n \n Roll no %d",s[i].roll);
printf("\n name : %s",s[i].name);
printf("\n marks : %d ",s[i].marks);
}
getch();
}
Output :
Enter information of students:

For roll no1,
Enter name: Tom
Enter marks: 98

For roll no2,
Enter name: Jerry
Enter marks: 89 . . .

Displaying Information:

Name: Tom
Marks: 98
Roll no: 2
Name: Jerry
Marks: 89
