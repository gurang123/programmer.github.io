Add two number

In this program, user is asked to enter two integers. Two integers entered by the user is stored in variables no1 and no2 respectively. This is done using scanf() function.
if a user will input two numbers as : '5', '10' then '15' (5 + 10) will be printed.
Then, variables no1 and no2 are added using + operator and the result is stored in sum.
* write program to add two number.
Input :
#include<stdio.h>
#include<conio.h>
void main()
{
int no1,no2,sum;
clrscr();
printf("enter two value : ");
scanf("%d %d",&no1,&no2);
sum=no1+no2;
printf("addition is : %d",sum);
getch();
}
Output :
enter two value :5
10
addition is : 15
*  If you want to subtraction in above program you can change addition sign to substract sign.
*  If you want to multiplication than you can change addition sign to multiplication sign.
*  If you want to division than you can change addition sign to division sign.

swap two number

In the belove program, the temporaryVariable is assigned the value of no1. Then, the value of no1 is assigned to no2.
Finally, the temporaryVariable (which holds the initial value of no1) is assigned to no2 which completes the swapping process.
* write program to swap two value with use of third variable.
Input :
#include<stdio.h>
#include<conio.h>
void main()
{
int no1,no2,temp;
clrscr();
printf("enter value of no1 : ");
scanf("%d",&no1);
printf("enter value of no2 : ");
scanf("%d",&no2);
temp=no1;
no1=no2;
no2=temp;
printf("after swap no1 is : %d",no1);
printf("after swap no2 is : %d",no2);
getch();
}
Output :

enter value of no1 :5
enter value of no2 :10

after swap no1 is : 10
after swap no2 is : 5
