Pointer program in c

A pointer is a variable that stores the address of another variable. Unlike other variables that hold values of a certain type, pointer holds the address of a variable.
For example, an integer variable holds a integer value, however an integer pointer holds the address of a integer variable.

Pointer Syntax

int* p;
* A Simple Example of Pointers in C.
Input :
#include<stdio.h>
#include<conio.h>
void main()
{
// Variable declaration
int num = 10;

// Pointer declaration
int *p;
clrscr();
// Assigning address of num to the pointer p
p = &num

printf("Address of variable num is: %p", p);

getch();
}
Output :
Address of variable num is: FFF4

Swap two numbers using pointer
* C program to swap two numbers using pointers.
Input :
#include<stdio.h>
#include<conio.h>
void swap(int *a,int *b)
{
int t;
t = *a;
*a = *b;
*b = t;
}
void main()
{
int num1,num2;
clrscr();
printf("Enter value of num1: ");
scanf("%d",&num1);
printf("Enter value of num2: ");
scanf("%d",&num2);
printf("Before Swapping: num1=%d, num2=%d\n",num1,num2);
swap(&num1,&num2);
printf("After Swapping: num1=%d, num2=%d\n",num1,num2);
getch();
}
Output :
Enter value of num1: 10
Enter value of num2: 20
Before Swapping: num1=10, num2=20
After Swapping: num1=20, num2=10

C program for double pointer (pointer to pointer)
* C program to demonstrate example of double pointer (pointer to pointer).
Input :
#include<stdio.h>
#include<conio.h>
void main()
{
int a;
int **p2;
p1=&a;
p2=&p1;
a=100;
clrscr();
printf("\nValue of a (using p1): %d",*p1);
printf("\nValue of a (using p2): %d",**p2);
*p1=200;
printf("\nValue of a: %d",*p1);
**p2=200;
printf("\nValue of a: %d",**p2);
getch();
}
Output :
Value of a (using p1): 100
Value of a (using p2): 100
Value of a: 200
Value of a: 200

C program for array of pointers
*C program to demonstrate example of array of pointers.

{
